АИС "СвОД"

Автоматизированная информационная система "Сведения Оперативных Данных". Содержит перечень подсистем, таких как:

Проверка сигнала, Строевая записка, План работы ССГ/Пикетов, Оперативно-профилактическая работа, Брошенные и оставленные поезда, Отцепы вагонов, Контроль ОД, База хищений и проишествий.

АИС "Учет собак"

Информационная система "Учет собак" предназначена внесения данных по служебным собакам. Имеется следующая информация:

Кличка собаки, Учетный номер(клеймо родословной), Инвертарный номер, Балансовая стоимость, Окрас, Дата рождения, Наличие документов, Отряд, За кем закреплена, Где содержится, Назначение и Место применения.

ИС по пожарной безопасности

Информационная система по пожарной безопасности объектов инфраструктуры ж.д.т. в границах ВСЖД. Содержит:

Предписания - обзор предписаний о приостановлении, Представления - обзор представлений, Инспектора - список инспекторов, Журнал ТПС - журнал учета огнетушителей при комиссионом осмотре локомотивов.

ИС учета разрешений

Информационная система по учету разрешений на хранение оружия. Содержит список разрешений, имеющих следующие параметры:

Отряд, Команда, ФИО, № Разрешения серии РХИ, № Разрешения серии РЛСа, Дата выдачи, Дата окончания действия разрешения, Дата направления документов в полицию на замену, Дата сдачи разрешения.
